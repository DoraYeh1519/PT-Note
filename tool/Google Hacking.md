
| 指令          | 功能            |
| ----------- | ------------- |
| `inurl:`    | 網址中包含關鍵字      |
| `intitle:`  | 標題包含關鍵字       |
| `intext:`   | 正文內容包含關鍵字     |
| `filetype:` | 搜尋特定檔案類型      |
| `site:`     | 限定網站範圍        |
| `link:`     | 找出連結到該網站的 URL |
