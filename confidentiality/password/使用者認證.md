 - 資訊安全基礎建設
	 - 基於存取控制及使用者責任制
 - 對於系統控管權，需有一套驗證使用者的程序，分為兩個步驟
	 - 識別：明確的使用者 ID
	 - 核實：將個體(使用者)與 ID 綁定，並確認該 ID 屬於所屬之使用者

## 認證方法分類（四大類）

| 類型                 | 範例           |
| ------------------ | ------------ |
| Something you know | 密碼、PIN、通行片語  |
| Something you have | 智慧卡、Token、手機 |
| Something you are  | 指紋、臉部、虹膜     |
| Something you do   | 輸入節奏、聲音波形、筆跡 |
