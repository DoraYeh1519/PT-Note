#### SOA
- Start Of Authority，網域管理資訊
#### NS : name server
- DNS伺服器主機名稱 
#### A/AAAA : IPv4 / IPv6 Address
- DNS網域名稱 對應到IPv4 / IPv6 Address  
#### CNAME : canonical name
- 主機名字的別名 
#### MX : mail exchanger
- 郵件傳輸代理（MTA, Message Transfer Agents） 
#### PTR : pointer
- IP 對應的 domain name  
#### TXT : 任意可讀的文字
